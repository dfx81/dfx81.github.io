22 June 2019 - Website Migration

Hello!

As you can see, the website have migrated to a brand new web host. The website no longer hosted by Github Pages, instead it's now hosted by Firebase Web Hosting. In addition to the host change, I even updated the site with a brand new look. It looks slicker and more professional than before.

There were two reason for this change. First of all, Firebase Web Hosting gave you two different URL for your site, which were .firebaseapp.com and .web.app. I'm hooked by the second URL, which is nice and simple. You don't have to type a long URL to access my site.

The final reason was the other features offered by Firebase itself. With features such as Firebase Cloud Functions, Firebase Realtime Database and Firebase Cloud Storage, there's even more stuff possible. For example, I can start making multiplayer games using Realtime Database and integrate my programs with Firebase Authentication. Features like Leaderboards are even possible due to Firebase. You just have to wait for what's coming in the future.

That's all for this current update. I don't know what to do with the old site on Github Pages. It maybe will redirect to the new site. Anyway, stay tuned for any new updates.
