Snake VS

Snake VS is a the same classic snake game that everyone loves. Popularized by its inclusion with Nokia 3310, this fanmade version intends to relive the old high-score based competitive spirit with modern technology. Now, every scores are collected into a leaderboard so everyone can compete for the top spot.
